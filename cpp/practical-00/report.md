# Practical 00 Report

## Overview
I completed all required questions in Practical 00 by implementing each task in a single C++ program. I structured the code into clear sections for each question and reused helper functions (like digit sum, prime check, and string utilities) so the logic is easier to follow and maintain.

## Implementation Notes (My POV)
I started with basic input/output for the personal introduction section, then moved to arithmetic operations based on my student number. For string manipulation, I wrote small functions to convert case, count vowels, reverse text, and extract initials. These functions made the later questions cleaner because I could call them instead of repeating logic.

For the user input and grade classification parts, I used simple input prompts and conditional checks to calculate the year of birth and assign grades. I followed this with loop-based patterns to practice repetition and nested loops. For array operations, I stored marks in a fixed array, computed highest, lowest, and average, and also visualized them using simple bars.

To practice modular programming, I grouped repeated logic into functions and displayed their outputs together. Then I worked with a vector to store user inputs, sort them, and calculate statistics like min, max, sum, and average. Finally, I implemented a `Student` class to apply OOP concepts and printed a formatted academic report.
